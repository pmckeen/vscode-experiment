# vscode-experiment

A small monorepo for a handful of experiments:

- `apps/home`: landing page for the browser experiments
- `apps/cryptosolver`: CryptoQuip decoder web app
- `apps/stats-fixer`: Ingress stats TSV correction tool
- `tools/cryptoquip-cli`: Python prototype for decoding CryptoQuips

Each browser app is self-contained with its own `index.html`, `styles.css`, and `script.js`.
